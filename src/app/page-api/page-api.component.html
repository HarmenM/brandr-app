BRANDr API

The BRANDr API may be used free of charge within the bounds of fair use for non-commercial purposes. Would you like to use BRANDr in your commercial app? Check out the BRANDr API git repository and host it yourself. BRANDr is shipped in a Docker container and can be deployed anywhere.

API Request

Check out the API with your favorite REST request tool, such as Postman.

Fire a x-form-urlencoded POST request to BRANDr endpoint

Endpoint {{apiUrl}}
Method POST
Body x-form-urlencoded form parameters
- endpoint <<website URL to retrieve>>
POST /extract/ HTTP/1.1
Host: <<BRANDr host>>
Content-Type: application/x-www-form-urlencoded
Accept: */*
accept-encoding: gzip, deflate

endpoint=<<URL encoded URL to retrieve>>
